# TERGM_global_arms_trade_1993-2023

Modelling the post-Cold War network of international major conventional weapons transfers using SIPRI data.

Code files accompanying the paper "The International Arms Transfers Network 1993-2023: A Sequence of Temporal Exponential Random Graph Models", QN Nguyen 2025 (0_Nguyen_2025_pCW_arms_network.pdf)

Objects outputs from the scripts were regularly saved as separate .RData files. The most relevant ones are found in the "number_objects" folders. Less important ones were not uploaded, but are available upon request.
